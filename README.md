# 柚子杯黑客马拉松赛事操作手册
#### 1.0版

为更好的举办各地黑客马拉松巡回赛，特制定本操作手册。

本手册主要用于指导城市合伙人筹办各分站赛事。

以下是城市合伙人与官方各自需要准备的事项：

### 一、场地

1- 活动场地需要能通宵开放使用，并有一定的安全保卫设施。

2- 公共场地约200-500平米，并有5-6间相对独立的10-20平方米会议室，供团队内部讨论与工作。

3- 有基本的路演设备，如投影仪、音响设备等。

4- 确保WIFI信号佳。

5- 有24小时空调设备，通风状况良好。

6- 有饮水机、咖啡机等基本茶水设备。

7- 周边有方便的外卖等商业设施。

8- 有沙发等休息设施以及减压设施优先考虑。

以上场地有城市合伙人准备，并承担所有费用，官方有权审核场地是否符合标准。

### 三、活动流程
**周六上午：**

|日期|时间|活动|备注|
|--|--|--|--|
|周六|9:00-9:30|签到||
||9:30-9:40|主持人开场||
||9:40-10:00|嘉宾发言|1名嘉宾，1名城市合伙人|
||10:00-10:15|公布比赛规程|包括比赛方式、比赛流程、评选方式、奖品，参与方介绍等等|
||10:15-10:45|参赛者自我介绍|每人一分钟|
||10:45-11:30|课题介绍|由主办方或者赞助商讲述业务需求与痛点|
||11:30-12:00|组队与团建|根据课题组成3-5人/组的战队|
||12:00-13:00|午餐||
||13:00-24:00|**正赛**|期间 18:30-19:30 供应晚餐<br>正赛分四个阶段，见下表|
|周日|00:00-12:00|**正赛**|期间 7:30-8:30 供应早餐|
||12:00-13:00|午餐||
||13:00-14:00|最后冲刺||
||14:00-15:30|成果展示与路演|每个团队限定10分钟，加5分钟提问|
||15:30-16:00|评委投票||
||16:00-16:30|颁奖、合影||
||16:30-17:00|交流时间||

**正赛阶段：**

|阶段|内容|
|--|--|
|一|完成团建活动|
|二|头脑风暴并形成初步方案，并用不超过300字说明|
|三|完成智能合约部署|
|四|完成前端UI|
|五|完成核心逻辑|
|六|完成演示PPT|
|七|完成产品原型|

**每一阶段结束后，需手持计时卡，拍照，以对完成进度进行确权。同时根据完成时间进行打分。**

### 四、志愿者
以下事务由志愿者负责。

城市合伙人需要合理安排志愿者的工作，做到责任到人，劳逸结合。

|岗位|职责|备注|
|--|--|--|
|签到|负责签到参赛者||
|主持人|负责主持，并把控所有环节时间与进度|
|赛前物资准备|负责以上物资准备工作，已经现场各项物资管理|
|团队辅助|每人负责2个团队，负责记录团队进度|需过夜|
|评委工作组|负责统计计时分数，并根据[《白皮书》](https://github.com/eoshackathon/hackathon_whitepaper)要求，辅助评委完成评审工作||
|财务组|管理人民币与数字货币账户，<br>负责按[《白皮书》](https://github.com/eoshackathon/hackathon_whitepaper)统计并记录各参与方的积分奖励||
|媒体组|负责联系媒体发通告，以及维护网站上的内容||
|直播设备|负责调试直播设备，确保赛事全程直播正常进行|需过夜|
|摄影|负责现场拍摄|需过夜|

### 五、开发者报名
开发者报名由城市合伙人与官方共同组织。

城市合伙人负责将场地情况向官方回报，经审核后，由官方负责发布报名信息。

### 六、评委
按[《白皮书》](https://github.com/eoshackathon/hackathon_whitepaper)要求，每个分站，由城市合伙人负责物色三名专业评委。

评委确定后，需要将评委信息（包括照片、简介）发给官方。
